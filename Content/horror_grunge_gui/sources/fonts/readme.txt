1. Download the Londrina Solid font from https://fonts.google.com/specimen/Londrina+Solid

2. Place the downloaded font files in the 'Content/horror_grunge_gui/sources/fonts' directory using File Explorer. If Unreal Engine prompts to auto-import new files, dismiss the popup.

3. Navigate to 'Content/horror_grunge_gui/fonts' and double-click the 'f_Font1.uasset'. In the Composite Font Tab, update the source file by clicking the folder icon and selecting the corresponding font files from the 'fonts'. Save changes.
